# 🌿 GreenPulse AI

Real-Time Urban Air Quality Intelligence & Alert System.

## Features
- Pathway real-time streaming
- AI risk prediction
- FastAPI backend
- Streamlit dashboard

## Structure
```
app/ dashboard/ config/ scripts/ tests/ docs/
```

## Run
```
pip install -r requirements.txt
python scripts/run_all.py
streamlit run dashboard/dashboard.py
```
